### 프로젝트 개요

바닥에 점으로 된 패턴(가칭: l-code) (전역좌표계)을 도포하여 그 위를 로봇이 지나다니면서 부분적으로 코드를 스캔, 자신의 위치를 2차원 좌표계로 인식

### 로봇 환경

* Yahboom의 raspbot v2를 사용
* 라이다, 엔코더 모터, IMU 없음 (노란색 TT 모터)
* 예제 파이썬 코드에서 속도를 10으로 했을 때 1초 동안 약 12.5cm를 이동한다는 측정 결과를 사용하여 엔코더를 대체

### 개발 환경

* docker.io/osrf/ros:humble-desktop-full 이미지를 사용하여 컨테이너 환경에서 ROS로 개발
* ROS2의 nav2 기능을 사용

### 점 패턴 디코더 관련

* 바닥의 점 패턴을 인식하는 디코더는 C++로 작성 (l-code)
* 디코더는 인식한 2차원 좌표와 로봇이 바라보는 방향을 반환
* 점 패턴의 좌표계는 좌상단이 (0,0), 우하단이 (550, 550)

### 전체적인 작동 방식

* 로봇 실행
* 스마트폰과 로봇 연결
    * 라즈베리파이의 핫스팟으로 연결
* 스마트폰 앱에서 로봇의 IPv4 주소와 포트번호 입력
* 스마트폰 앱에 표시된 좌표계에서 사용자가 원하는 타겟(목적지) 좌표 터치
* 사용자가 터치한 좌표를 로봇으로 전송
* 로봇은 nav2를 사용하여 사용자가 지정한 좌표로 이동
    * 이 과정에서 점 패턴 디코더 사용하여 로봇은 자신의 위치를 지속적으로 인지
    * 엔코더 모터가 없으므로 위에서 설명한 속도 10에 1초 동안 12.5cm를 이동한 실측값 사용
* 로봇이 이동하면서 스마트폰 앱으로 자신의 위치를 지속적으로 전송
* 스마트폰은 로봇의 현재 위치를 화면에 표시
